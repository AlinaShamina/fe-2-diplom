## О проекте

Сервис покупки железнодорожных билетов.

Реализовано:

* поиск поездов
* фильтрация маршрутов
* выбор типа вагона
* выбор мест
* схема вагонов
* оформление пассажиров
* страница оплаты
* адаптивная верстка
* публикация на GitHub Pages
